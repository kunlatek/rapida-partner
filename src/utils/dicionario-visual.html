Exemplo de Documentação: Lista (IList)

Instrução: Monte sua lista assim. Comece com os metadados (wrapper, datasource, etc.) e depois defina o "Template do Item" empilhando as propriedades na ordem em que devem aparecer. 
componentType: list
title: Lista de Personagens
dataSource: {
  endpoint: "/api/characters",
  paramType: "query"
}
cardAsALink: {
  link: "/personagem-detalhe",
  propertiesAsPathParam: ["_id"]
